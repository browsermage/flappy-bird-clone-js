<!-- STARTING -->
## Getting started 

1. Clone the repo
   ```sh
   git clone https://github.com/colorsofcode/flappy-bird-clone-js.git
   ```
2. Move to the new directory
    ```sh
    cd flappy-bird-clone-js
    ```
3. Start up your the development environment
   ```sh
   npx http-server
   ```
